# Help & Support

Need help with LedgeIT? Find quick answers for setup, usage, troubleshooting, and support contact.

## Technical Questions

### What versions are supported?

LedgeIT supports recent iOS versions available through the App Store. For the best experience, keep both iOS and LedgeIT updated to the latest release.

### Can I schedule automatic tracking actions?

LedgeIT supports reminders and recurring financial entries where applicable. Availability may vary by module and app version.

### Does LedgeIT affect device performance?

LedgeIT is designed for efficient, local-first operation with minimal performance impact during normal use.

## Getting Started

### First Launch

When you open LedgeIT for the first time, complete onboarding and profile setup so calculations, goals, and dashboards match your context.

### Granting Permissions

LedgeIT may request permissions such as notifications, or files access (for imports/exports), depending on features you use.

### Add your first records

Start with income, expenses, and savings entries. Then add investments, assets, debt, insurance, and goals for a complete financial view.

### Import existing data

If you are migrating from spreadsheets or another app, use CSV import where supported and verify category mappings before finalizing.

## Troubleshooting

### App not launching

Restart your device and confirm you are on the latest App Store build. If the issue continues, contact support on ledgeit.app@gmail.com with device model and iOS version details.

### Permission issues

Check iOS Settings for LedgeIT permissions and re-enable required access. Reopen the app after changing permission settings.

### Slow performance

Close heavy background apps, ensure available device storage, and update to the latest app version for performance improvements.
